用nginx搭建谷歌镜像网站

发表于 2016-10-29   |   分类于 技术教程   |  

1. 介绍

搭建类似http://tokillgoogle.com/这样的网站，只是能让我们访问google.com。用的工具是ngx_http_google_filter_module，是一个nginx的插件，用的原理是nginx的反向代理。

2. 编译安装

首先要有一台能访问google.com的vps或云主机，并且确保之前编译安装过nginx。

这个插件依赖于ngx_http_substitutions_filter_module这个库。

<span class="gp">$ </span>git clone https://github.com/cuber/ngx_http_google_filter_module
<span class="gp">$ </span>git clone https://github.com/yaoweibin/ngx_http_substitutions_filter_module
`</pre>
<pre class="highlight shell">`<span class="gp">$ </span><span class="nb">cd </span>nginx
<span class="gp">$ </span>./configure <span class="se">\</span>
--user<span class="o">=</span>nginx                          <span class="se">\</span>
--group<span class="o">=</span>nginx                         <span class="se">\</span>
--prefix<span class="o">=</span>/etc/nginx                   <span class="se">\</span>
--sbin-path<span class="o">=</span>/usr/sbin/nginx           <span class="se">\</span>
--conf-path<span class="o">=</span>/etc/nginx/nginx.conf     <span class="se">\</span>
--pid-path<span class="o">=</span>/var/run/nginx.pid         <span class="se">\</span>
--lock-path<span class="o">=</span>/var/run/nginx.lock       <span class="se">\</span>
--error-log-path<span class="o">=</span>/var/log/nginx/error.log <span class="se">\</span>
--http-log-path<span class="o">=</span>/var/log/nginx/access.log <span class="se">\</span>
--with-http_gzip_static_module        <span class="se">\</span>
--with-http_stub_status_module        <span class="se">\</span>
--with-http_ssl_module                <span class="se">\</span>
--with-pcre                           <span class="se">\</span>
--with-file-aio                       <span class="se">\</span>
--with-http_realip_module             <span class="se">\</span>
--without-http_scgi_module            <span class="se">\</span>
--without-http_uwsgi_module           <span class="se">\</span>
--without-http_fastcgi_module         <span class="se">\</span>
--add-module<span class="o">=</span>/home/ubuntu/softs/ngx_http_google_filter_module <span class="se">\</span>
--add-module<span class="o">=</span>/home/ubuntu/softs/ngx_http_substitutions_filter_module <span class="se">\</span>
`</pre>
具体的编译参数可以通过`nginx -V`查到。

`--add-module`指定插件的保存位置。

接下来编译安装。
<pre class="highlight shell">`<span class="gp">$ </span>make
<span class="gp">$ </span>sudo make install
`</pre>
重启服务。
<pre class="highlight shell">`<span class="gp">$ </span>sudo make upgrade
`</pre>
还可以用`nginx -V`查看是否编译成功。

#### 3\. 配置使用

打开配置文件`/etc/nginx/nginx.conf`。
<pre class="highlight conf">`<span class="n">server</span> {
  <span class="c"># ... part of server configuration
</span>  <span class="n">resolver</span> <span class="m">8</span>.<span class="m">8</span>.<span class="m">8</span>.<span class="m">8</span>;
  <span class="n">location</span> / {
    <span class="n">google</span> <span class="n">on</span>;
  }
  <span class="c"># ...
</span>}
`</pre>
找到server部分，添加`resolver`和`location`两个指令，总共四行。

让配置文件生效。
<pre class="highlight shell">`<span class="gp">$ </span>sudo nginx -s reload

成功，看到效果。
